# ramadhan-rush
Ramadhan Rush adalah sebuah web-game arcade interaktif bergaya WarioWare yang mengangkat budaya dan hiruk-pikuk bulan puasa di Indonesia. Pemain akan diuji ketangkasan dan fokusnya melalui rentetan mini-game bertempo cepat yang akan terus meningkat kesulitannya seiring berjalannya waktu.

Dibuat menggunakan React dan HTML5 Canvas, game ini menghadirkan pengalaman visual yang mulus, efek suara retro 8-bit yang nostalgia, serta humor lokal yang sangat relatable bagi masyarakat Indonesia.
